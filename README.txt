README

=== csv2graph ===

Description
-----------------
The name of this project is csv2graph, as the name suggests, it transforms csv files (old beijing text file) data to graph data structure for the convenience of further utility and analysis.

Users can change the text file name of Node and Road in BuildGraph class. 

When the program is executed, it will extract useful information from the designated text file and build up a graph. All vertexes and edges in the graph will be printed out, and a shortest path from source vertex to target vertex,
assigned by user, will be calculated as well.
